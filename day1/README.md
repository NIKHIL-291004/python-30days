## üñ•Ô∏è Computer System Fundamentals

### Core Components of a Computer

A computer system consists of the following major components:

#### 1. Central Processing Unit (CPU)

The CPU is the **brain of the computer** and performs all processing tasks.

**Internal components of CPU:**

* **ALU (Arithmetic Logic Unit):** Performs arithmetic and logical operations (AND, OR, XOR, etc.)
* **Control Unit:** Directs and controls all operations in the system
* **Clock:** Synchronizes all CPU operations
* **Registers:** Small, high-speed storage units inside the CPU used for immediate processing

#### 2. Input / Output Devices

* **Input Devices:** Mouse, Keyboard, Scanner, etc.
* **Output Devices:** Monitor, Printer, Speakers, etc.

#### 3. Memory Units

* **RAM (Random Access Memory):** Volatile memory; data is lost when power is off
* **ROM (Read Only Memory):** Non-volatile memory; data is retained even after power off

---

## üß† Memory Hierarchy

The computer uses different types of memory arranged in a hierarchy based on **speed and cost**:

**Fastest ‚Üí Slowest**

1. Registers
2. Cache Memory
3. Main Memory (RAM)
4. Secondary Memory (HDD/SSD, DVD, Magnetic Tape)

### Key Points:

* The CPU can operate **only on data present in registers**
* Faster memory is more expensive
* Slower memory is cheaper but provides large storage
* Data flows from Secondary Memory ‚Üí RAM ‚Üí Cache ‚Üí Registers

## üí° What is Programming?

* **Programming / Coding** is the process of **communicating with the processor**
* The processor is a non-living entity and cannot understand human languages directly
* A **Programming Language** acts as a medium between humans and machines

### Program & Software

* **Program (Code):** A set of instructions given to the processor
* **Software:** A collection of programs

---

## ‚öôÔ∏è Processor Language & Compilation

### Processor Language

* Internally, the processor understands only **machine language**
* Machine language consists of **binary values (0s and 1s)**
* These represent **Low Voltage / High Voltage** or **False / True**

### Why Conversion is Needed?

* Humans write programs using **high-level languages** (English-like syntax)
* The processor understands only **machine language**

### Compilation Process

1. High-Level Language Code
2. Assembly Language
3. Machine Language (0s and 1s)

This conversion process is known as **Compilation**.

---

## üåê Language Hierarchy

### Types of Programming Languages

1. **Low-Level Language**

   * Machine Language (0s and 1s)
   * Very fast but difficult for humans

2. **Medium-Level Language**

   * Assembly Language
   * Uses mnemonics

3. **High-Level Language**

   * Python, C, Java, etc.
   * Easy to read and write

---

## üîÑ Compiler vs Interpreter

| Feature        | Compiler                            | Interpreter              |
| -------------- | ----------------------------------- | ------------------------ |
| Translation    | Entire program at once              | Line by line             |
| Error Handling | Errors shown after full compilation | Errors shown immediately |
| Speed          | Faster execution                    | Slower execution         |

* **Python is an interpreted language**
